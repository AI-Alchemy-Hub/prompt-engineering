
# Prompt Engineering

> A curated repository of prompt libraries, templates, and pedagogical resources for Generative AI.  
> Part of **AI Alchemy Hub**.

Prompt Engineering provides structured examples, reusable templates, and best practices for designing effective prompts across diverse AI systems. It is intended for learners, educators, and professionals seeking clarity and reproducibility in prompt design.

## Contents
- `libraries/` → reusable prompt collections organized by domain
- `templates/` → starter templates for workshops and projects
- `examples/` → annotated examples demonstrating prompt strategies
- `docs/` → supporting documentation and GitHub Pages site

## Getting Started

Clone the repository and set up your environment:

    git clone https://github.com/AI-Alchemy-Hub/prompt-engineering.git
    cd prompt-engineering
    python -m venv venv
    source venv/bin/activate   # Linux/macOS
    venv\Scripts\activate      # Windows
    pip install -r requirements.txt
